# Matrix-Operations-Tool
Matrix Operations Tool using NumPy
## Project Overview
An interactive Matrix Operations Tool built using 
Python and NumPy that performs mathematical 
operations on matrices.

## Domain
Python Development

## Internship
QSkill Internship | April 2026

## Libraries Used
- NumPy

## Operations Supported
| Operation | Description |
|-----------|-------------|
| Addition | Adds two matrices |
| Subtraction | Subtracts two matrices |
| Multiplication | Multiplies two matrices |
| Transpose | Flips rows and columns |
| Determinant | Calculates determinant value |

## How to Run
1. Open Google Colab
2. Upload Matrix_Operations_Tool.ipynb
3. Run Cell 1 → Import
4. Run Cell 2 → Helper Functions
5. Run Cell 3 → Operations
6. Run Cell 4 to 8 → Use any operation

## How to Enter Matrix
Each operation asks you to enter like this:
- Number of rows
- Number of columns
- Values row by row
